# Galactic Travel Agency (C#)

This C# console application demonstrates basic programming concepts by simulating a simple galactic travel agency system. It stores and displays passenger information, updates the passenger’s age, and showcases different types of data type conversions.

## Features
- Stores passenger details (name, age, ticket type, preferred planet)
- Displays formatted output to the console
- Updates the passenger’s age
- Demonstrates:
  - Explicit type conversion
  - Implicit type conversion
  - Conversion using helper methods

## Concepts Demonstrated
- Variables and data types
- Console output formatting
- Arithmetic operations
- Explicit vs. implicit type conversions

## How to Run
1. Save the file as `Program.cs`
2. Compile using:
   ```bash
   dotnet build
   ```
 3. Run using:
    ```bash
    dotnet run
    ```
